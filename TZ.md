# Техническое задание: Сервис подбора удобрений для комнатных растений

## Идея проекта
Веб-сервис помогает пользователю подобрать удобрение для комнатного растения, рассчитать дозировку и получить график подкормок на месяц вперёд. Сервис учитывает фазу роста растения и автоматически рекомендует подходящий тип удобрения.

## Роли пользователей
- **Гость** — может просматривать каталог растений, искать растения, подбирать удобрения, рассчитывать дозировку и просматривать график подкормок.
- **Администратор** — через встроенную админку Django управляет справочниками растений и удобрений.

## Основные сценарии использования

### Сценарий 1: Подбор удобрения
1. Пользователь заходит на главную страницу
2. Видит каталог растений с поиском
3. Нажимает на интересующее растение
4. Видит рекомендуемое удобрение (автоподбор по фазе роста)
5. Выбирает удобрение из списка
6. Вводит объём воды для полива
7. Нажимает «Рассчитать»
8. Получает точную дозировку (мл) и график подкормок

### Сценарий 2: Поиск растения
1. Пользователь вводит часть названия в строку поиска
2. Система показывает только подходящие растения
3. Поиск не зависит от регистра

### Сценарий 3: Администрирование
1. Администратор заходит в админку
2. Может добавлять, редактировать и удалять растения и удобрения
3. Может искать и фильтровать записи

## Схема данных (3 кастомные модели)

### Plant (Растение)
| Поле | Тип | Описание |
|------|-----|----------|
| name | CharField(100) | Название растения |
| latin_name | CharField(100) | Латинское название |
| description | TextField | Описание |
| growth_phase | CharField(20) | Фаза: рост / цветение / покой |
| feeding_interval_days | PositiveIntegerField | Интервал между подкормками |

### Fertilizer (Удобрение)
| Поле | Тип | Описание |
|------|-----|----------|
| name | CharField(100) | Название удобрения |
| type | CharField(20) | Тип: минеральное / органическое / универсальное |
| npk_ratio | CharField(20) | Состав NPK (например, 10-10-10) |
| concentration_ml_per_liter | DecimalField | Концентрация (мл на 1 л воды) |
| description | TextField | Описание |

### FeedingSchedule (График подкормки)
| Поле | Тип | Описание |
|------|-----|----------|
| plant | ForeignKey → Plant | Связь с растением |
| fertilizer | ForeignKey → Fertilizer | Связь с удобрением |
| start_date | DateField | Дата начала |
| dosage_ml | DecimalField | Дозировка (мл) |
| next_feeding_date | DateField | Дата следующей подкормки |

## Расчёт дозировки
Дозировка (мл) = Концентрация удобрения (мл/л) × Объём воды (л)

График строится на 8 подкормок вперёд с шагом, равным интервалу подкормки растения. Дозировка увеличивается на 5% с каждой следующей подкормкой.

## Используемые технологии
- **Backend:** Django 6.x, Django ORM
- **Frontend:** HTML5, Bootstrap 5 (CDN)
- **Визуализация:** Plotly (столбчатая диаграмма)
- **Обработка данных:** Pandas
- **База данных:** SQLite (разработка), MySQL (на PythonAnywhere)
- **Деплой:** PythonAnywhere
